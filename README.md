# transparent-shell

Минималистичная тема для **GNOME Shell** с полупрозрачными панелями и аккуратной стилизацией системных элементов интерфейса.

Тема делает фон меню, быстрых настроек, уведомлений и других всплывающих элементов полупрозрачным — обои рабочего стола остаются видны сквозь интерфейс, сохраняя читаемость текста и контраст.

## Возможности

- **Полупрозрачные панели** — меню, быстрые настройки, баннеры уведомлений, OSD-индикаторы, модальные окна и переключатель окон
- **Календарь** — скруглённые углы, плавные hover/focus-состояния, выделение текущего дня цветом акцента системы
- **Ползунки** — кастомная высота и цвета для регуляторов громкости и яркости
- **Полосы прокрутки** — тонкие, ненавязчивые скроллбары с hover-эффектами
- **Совместимость** — CSS содержит селекторы для GNOME Shell 44–46+

## Требования

- GNOME Shell **44** или новее (рекомендуется 45–46)
- Расширение [GNOME Tweaks](https://wiki.gnome.org/Apps/Tweaks) — для выбора темы через графический интерфейс (необязательно, можно через `gsettings`)

## Установка

### Из репозитория

```bash
git clone https://github.com/GlebKosyukov/transparent-shell.git ~/.themes/transparent-shell
```

### Вручную

Скопируйте папку `transparent-shell` в каталог тем:

```bash
cp -r transparent-shell ~/.themes/
```

Для установки для всех пользователей системы:

```bash
sudo cp -r transparent-shell /usr/share/themes/
```

## Применение

### Через GNOME Tweaks

1. Откройте **GNOME Tweaks** → раздел **Appearance** / **Внешний вид**
2. В поле **Shell** выберите **transparent-shell**

### Через терминал

```bash
gsettings set org.gnome.shell.extensions.user-theme name 'transparent-shell'
```

> Для применения через `gsettings` должно быть установлено и включено расширение [User Themes](https://extensions.gnome.org/extension/19/user-themes/).

После смены темы перезапустите GNOME Shell:

- **Wayland:** `Alt+F2` → `r` → Enter (или выход из сеанса и повторный вход)
- **X11:** `Alt+F2` → `r` → Enter

## Структура

```
transparent-shell/
└── gnome-shell/
    └── gnome-shell.css    # Основной файл стилей
```

Тема наследует базовые стили системной темы GNOME и переопределяет только отдельные элементы — установка не затрагивает GTK-приложения и иконки.

## Настройка

Все параметры задаются в `gnome-shell/gnome-shell.css`. Палитра цветов перечислена в комментарии в начале файла. Основной цвет фона панелей:

```css
background: rgba(32, 32, 32, 0.75);
```

Измените значения `rgba(...)` для настройки оттенка и степени прозрачности. После правок перезапустите GNOME Shell.

## Известные ограничения

- В CSS используются ссылки на `calendar-event.svg` и `calendar-event-today.svg` для отметки дней с событиями — эти файлы пока не включены в репозиторий, поэтому индикаторы событий в календаре могут не отображаться.
- Тема оформляет только **GNOME Shell** (верхняя панель, меню, уведомления). Для окон приложений используйте отдельную GTK-тему.

## Удаление

```bash
rm -rf ~/.themes/transparent-shell
gsettings reset org.gnome.shell.extensions.user-theme name
```
